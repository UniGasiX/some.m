## Tag

有的“项目”可能能在某种意义上达到足够的“完成度”形成可以“释出”的“版本”。当有某“项目”在某次提交中达到了这种程度，将在该次提交上打上附注标签。该附注标签的标签名遵循该格式：```项目名-版本号```，其中，“项目名”为该“项目”的文件夹的名字，版本号为小写字母v开头的版本号。
